SQL Session 13

Advanced SQL queries and exercises.

Example 1

Find users whose salary is above the average salary.

SELECT name
FROM users
WHERE salary > (
    SELECT AVG(salary)
    FROM users
);

Example 2

Select distinct countries from the table.

SELECT DISTINCT country
FROM users;

Example 3

Find top 10 users by followers.

SELECT user_id, followers_count
FROM users
ORDER BY followers_count DESC
LIMIT 10;
